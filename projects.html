Virtual Closet

The Virtual Closet is a personal project built using Flask, designed to help users organize and manage their wardrobe digitally. Users can register, log in, and upload images of their clothing items, categorizing them by type, color, and other details. Each user has their own virtual closet where they can edit, update, or delete items. The platform includes features like profile customization with profile pictures and a user directory to view all registered users.

This project highlights full-stack web development skills, including backend development with Flask, database management with SQLAlchemy, and image handling for a user-friendly experience.

Technologies Used: Flask, SQLAlchemy (with SQLite), Python, JavaScript, HTML, CSS
